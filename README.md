# mcsr-utils

Miscellaneous scripts useful for Minecraft speedrunning

## Jingle-OBS Scene Manager

Extend [Jingle](https://github.com/DuncanRuns/Jingle) scripts to automatically adjust OBS scene visibility when using resizing scripts.
- Resizing (Custom).lua - Fork of Jingle's resizing script that additionally persists state info when resizing
- scene-mux-manager-lua - OBS script that monitors the above state and adjusts source visibility based on it
